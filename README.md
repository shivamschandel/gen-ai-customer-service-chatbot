# gen-ai-customer-service-chatbot

This project is developed as part of the **ElevanceSkills AI Internship**.

## Project Description
The goal of this project is to build an intelligent chatbot capable of assisting users with various queries using Natural Language Processing (NLP) and Generative AI techniques.

## Features
- Sentiment Analysis for detecting customer emotions
- Medical Question Answering using the MedQuAD dataset
- Dynamic knowledge base updates
- Scientific research assistant using arXiv dataset
- Multimodal chatbot (text + image understanding)
- Multilingual chatbot support

## Technologies Used
- Python
- Streamlit
- NLP (Natural Language Processing)
- HuggingFace Transformers
- Google Translate API
